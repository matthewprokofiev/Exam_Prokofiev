- **Тип проекта:** Приложение WPF (.NET Framework), версия .NET Framework 4.8.1
- **Решение:** `Exam` (проект `Exam` + тестовый проект `Exam.Tests`)

## Описание задания (Вариант 1)

Графическое приложение запрашивает баллы за каждый выполненный модуль,
суммирует их и выводит сумму набранных баллов и оценку по 5-балльной шкале.
Пользователь выбирает уровень экзамена:

- **Базовый (БУ)** — модули 1–3 (максимум 50 баллов)
- **Профильный (ПУ)** — модули 1–4 (максимум 75 баллов)
- **Профильный с вариативной частью (ПУ+)** — модули 1–5 (максимум 100 баллов)

Максимальные баллы по модулям: М1 = 10, М2 = 15, М3 = 25, М4 = 25, М5 = 25.

Шкала перевода процента выполнения в оценку:

| Процент выполнения | Оценка |
|--------------------|--------|
| 80%–100%           | 5 (отлично) |
| 60%–79%            | 4 (хорошо) |
| 40%–59%            | 3 (удовлетворительно) |
| 0%–39%             | 2 (неудовлетворительно) |
