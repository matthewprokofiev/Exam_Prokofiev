## Описание задания (Вариант 1)

Графическое приложение запрашивает баллы за каждый выполненный модуль,
суммирует их и выводит сумму набранных баллов и оценку по 5-балльной шкале.
Пользователь выбирает уровень экзамена:

- **Базовый (БУ)** — модули 1–3 (максимум 50 баллов)
- **Профильный (ПУ)** — модули 1–4 (максимум 75 баллов)
- **Профильный с вариативной частью (ПУ+)** — модули 1–5 (максимум 100 баллов)

Максимальные баллы по модулям: М1 = 10, М2 = 15, М3 = 25, М4 = 25, М5 = 25.

Шкала перевода процента выполнения в оценку:

| Процент выполнения | Оценка |
|--------------------|--------|
| 80%–100%           | 5 (отлично) |
| 60%–79%            | 4 (хорошо) |
| 40%–59%            | 3 (удовлетворительно) |
| 0%–39%             | 2 (неудовлетворительно) |

## Структура проекта

```
Exam/

    └── ExamLogicTests.cs      — автоматизированные тесты
```

## Архитектура (по единым требованиям)

- **WPF:** XAML-интерфейс, обработчики событий в code-behind
  (`buttonCalculate_Click`), валидация через `int.TryParse`,
  сообщения об ошибках через `MessageBox`. Приложение не вылетает.
- **Логика:** вынесена в отдельный статический класс `ExamLogic`. Методы
  принимают простые типы и возвращают результаты, при некорректных данных
  выбрасывают `ArgumentException`.
- **Тесты (MSTest):** отдельный проект, тестируются только методы `ExamLogic`.
  Используются `Assert.AreEqual`, `Assert.AreNotEqual`, `Assert.IsTrue`,
  `Assert.IsFalse`, `Assert.ThrowsException`. Покрыты корректные данные,
  граничные значения и некорректные данные (исключения).

## Как подключить файлы к уже созданному проекту

Если проект `Exam` уже создан в Visual Studio:

1. Скопируйте `ExamLogic.cs` в проект `Exam` (Добавить → Существующий элемент).
2. Замените содержимое `MainWindow.xaml` и `MainWindow.xaml.cs` на файлы из этой папки.
3. Добавьте тестовый проект: ПКМ по решению → Добавить → Новый проект →
   **Проект модульного теста (.NET Framework)** или используйте готовый
   `Exam.Tests` из этой папки.
4. В тестовом проекте добавьте ссылку на проект `Exam`
   (Зависимости/Ссылки → Добавить ссылку на проект → Exam).
5. Скопируйте `ExamLogicTests.cs` в тестовый проект.

Либо просто откройте `Exam.sln` из этой папки целиком.
